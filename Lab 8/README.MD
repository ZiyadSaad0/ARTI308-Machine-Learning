KNN Classification Project

Overview

Implementation of the K-Nearest Neighbors (KNN) algorithm using Scikit-learn for a classification task.

Workflow

Data loading with Pandas
Exploratory Data Analysis (EDA)
Feature scaling using StandardScaler
Train-test split (70/30)
Model training & prediction
Evaluation (Confusion Matrix, Precision, Recall, F1-score)
Optimal K selection using error rate
Results

Accuracy: ~72%
Performance improved after tuning K
Tools

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

Conclusion

KNN is simple and effective, but depends heavily on feature scaling and choosing the right K.
